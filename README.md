# BRITZMEDI 한국 홈페이지 (britzmedi.co.kr)

> 브리츠메디 한국 시장 B2B 홈페이지 — AEO/GEO 최적화 설계

## 기술 스택

| 기술 | 용도 |
|------|------|
| Astro 5 | SSG 프레임워크 |
| React 19 | 인터랙티브 컴포넌트 |
| Tailwind CSS 4 | 스타일링 |
| TypeScript | 타입 안전성 |
| Supabase | 인증, DB, CMS |
| Cloudflare Pages | 호스팅 + CDN |
| Claude API | AI 챗봇 |

## 시작하기

```bash
npm install
npm run dev
```

## 배포

```bash
npm run build
# Cloudflare Pages 자동 배포 (GitHub 연동)
```
